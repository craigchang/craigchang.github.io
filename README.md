## Tools & Stack

Built with:

- **Jekyll** — Static site generator; Markdown content and Liquid templates compiled into static HTML
- **GitHub Pages** — Hosting and deployment
- **SASS/SCSS** — Styling with modular partials (layout, components, highlights, etc.)
- **Kramdown** — Markdown parsing with GFM support
- **Liquid** — Templating for layouts, includes, and data-driven sections
- **jekyll-sitemap** — Sitemap generation for SEO

**Local development:**

```bash
bundle install
bundle exec jekyll serve
```
